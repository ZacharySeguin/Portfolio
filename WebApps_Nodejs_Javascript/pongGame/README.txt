*Code was developped in Windows 10, and tested in Google Chrome.*

1) Upon initial download of file, navigate to directory in terminal window and type in "npm install" to install modules. 
2) Then to start the server type in "node server.js". 
3) The server will host a webpage with URL: localhost:3000/assignment2.html
4) To test the game, open two tabs/browsers using the URL above (localhost:3000/assignment2.html) and follow the instructions displayed below the canvas.
